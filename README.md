## 🎯 Part A - Number Guessing Game

A simple guessing game where:
- The computer picks a random number between 1–100.
- You try to guess it in 10 attempts.
- Feedback is given: “Too High”, “Too Low”, or “Correct”.
- Sound effects and a timer are included.
- A new game starts after win/loss.

---

## 🧠 Part B - Memory Matching Game

A classic memory flip game:
- Choose 8, 10, or 12 pairs of cards.
- Select a difficulty: 3, 5, or 8 seconds to memorize.
- Cards flip back, and you must match all pairs before time runs out.
- Game includes scoring system and top 5 leaderboard saved with `localStorage`.

### 🏆 Scoring

| Action                  | Points |
|-------------------------|--------|
| Correct match           | +10    |
| Incorrect attempt       | -5     |
| Time exceeded penalty   | -1/sec |

---

## 🚀 How to Run

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Click either:
   - "Number Guessing Game" 
   - "Memory Matching Game" 

Works offline — no server required.
